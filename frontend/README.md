# Fly Bot Frontend

React + TypeScript frontend for the Fly Bot standby travel recommender.

## Prerequisites

- Node.js 18+ and npm
- Fly Bot API running (see [root README](../README.md))

## Quick Start

```bash
# Install dependencies
npm install

# Start the API server in demo mode (in a separate terminal, from repo root)
cd ..
make run-demo-bg

# Verify API is running
make demo-health

# Start the frontend dev server (from this directory)
npm run dev
```

Visit http://localhost:3000

## Development

```bash
# Run tests
npm test

# Run tests in watch mode
npm run test:watch

# Run tests with UI
npm run test:ui

# Lint
npm run lint

# Build for production
npm run build

# Preview production build
npm run preview
```

## Configuration

Create `.env.local` to override the API URL:

```
VITE_API_BASE_URL=http://your-api-server:8000
```
